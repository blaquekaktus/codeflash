What is a coroutine in Unity?
Coroutines spread logic over multiple frames using IEnumerator and yield. Use for delays, sequences, and async-like patterns.
IEnumerator SpawnWave() { for (int i = 0; i < 5; i++) { SpawnEnemy(); yield return new WaitForSeconds(0.5f); } }
godot
How do signals work in Godot?
Signals are Godot's observer pattern. Nodes emit signals, others connect. Decouples sender from receiver.
signal health_changed(new_health) func take_damage(amount): health -= amount health_changed.emit(health)
